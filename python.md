
## python dict
when print a dict, it looks like this 
```
{'food': 'pudding', 'quantity': 4, 'color': 'pink'}
```
To instance a dict
```
D = {'food': 'pudding', 'quantity': 4, 'color': 'pink'}
```
To fetch value
```
D['food']
```
